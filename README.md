- [링크]()

---

- 기능소개

1. 타이틀 작성
   </br>타이틀을 작성하면 입력값이 썸네일에 표시됩니다.

2. 랜덤 효과
   </br>버튼을 클릭하면 단색 배경이 랜덤으로 바뀝니다.
   </br>버튼을 클릭하면 그라디언트 배경이 랜덤으로 바뀝니다.
   </br>버튼을 클릭하면 이미지가 랜덤으로 바뀝니다.
   </br>버튼을 클릭하면 텍스트 색상이 랜덤으로 바뀝니다.
